Thoughts

My tag grid story is wonderful. I wanted to create tables showing relationships among tiddlers, based on two tags. For example, if I had four tiddlers, with the following tags:

Blue Honda Civic:: Blue, Honda, Civic Red Honda Fit:: Red, Honda, Fit Blue Subaru Forester: Blue, Subaru, Forester Red Subaru Outback: Red, Subaru, Outback

I'd want to be able to create this table:

RedBlue
HondaRed Honda FitBlue Honda Civic
SubaruRed Subaru OutbackBlue Subaru Forester

So, I posted in the TiddlyWiki Google Group, and Jed Carty created the TagGrid macro, which works beautifully:
