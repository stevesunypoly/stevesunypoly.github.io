As I get ready for THWT, I'm thinking of using the old scrapbook macros.

Functionality: Using the Firefox Extension Scrapbook, create a static representation of the page in its hypertextual context in an archival setting, using a timestamp as a unique identifier. The timestamp is embedded into a macro so that a link to the archived Web pages is easily created.

Technique: Put Scrapbook folder in Dropbox public folder. Link should be archived page
